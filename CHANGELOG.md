# Changelog

All notable changes to DevVault will be documented in this file.

## [0.1.1] - 2026-07-12

### Changed

- Marketplace icon moved to package root with stronger contrast for dark Open VSX / Cursor UI

## [0.1.0] - 2026-07-12

### Added

- Local encrypted credential vault for Cursor and VS Code
- Master password with AES-256-GCM encryption via PBKDF2
- Sidebar vault for browsing, adding, editing, and deleting entries
- Quick fill by URL/domain with clipboard auto-clear
- Password generator
- Idle lock and optional remembered unlock session
- Open in Browser & Login workflow for the built-in browser
- Status bar actions for the active credential entry
